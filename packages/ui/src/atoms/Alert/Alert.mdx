# Alert

화면 안에 인라인으로 표시하는 피드백 메시지 배너입니다.

## 언제 사용하나요?

- 로그인·제출 실패 등 오류를 안내할 때
- 참고 정보나 안내를 보여줄 때
- 작업이 성공했음을 알릴 때
- 주의가 필요한 상태(재고 부족 등)를 경고할 때

## Props

<Controls />

| Prop       | Description                                                                       |
| ---------- | -------------------------------------------------------------------------------- |
| `tone`     | 메시지 의미입니다. 색상·아이콘·통지 방식이 함께 바뀝니다.                          |
| `children` | 메시지 본문입니다. 블록 요소나 버튼도 넣을 수 있습니다.                            |
| `id`       | 요소 id 입니다. 폼 컨트롤의 `aria-describedby` 로 이 메시지를 가리킬 때 씁니다.    |
| `onClose`  | 제공하면 닫기(×) 버튼이 표시되고, 주지 않으면 표시되지 않습니다.                   |

## Behavior

- `onClose` 핸들러의 유무가 닫기 버튼의 유무를 결정합니다.
- 배너를 실제로 없애는 것(언마운트)은 사용하는 쪽의 책임입니다.
- 포커스를 옮길 때 쓰도록 `ref` 를 노출합니다(제출 실패 시 배너로 포커스 이동).

## Accessibility

- `danger` 는 즉시 통지, 그 외 tone 은 대기 통지로 안내됩니다.
- 색상만으로 의미를 전달하지 않도록 tone 별 아이콘을 함께 표시합니다.
- 배너 본문은 탭 순서에 들어가지 않고, 닫기 버튼은 `Tab`·`Enter`·`Space` 로 조작할 수 있습니다.
- 닫기 버튼에는 접근 가능한 이름이 제공됩니다.
