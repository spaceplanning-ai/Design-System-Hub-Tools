# Badge

제목이나 라벨 옆에 개수를 표시하는 비대화형 카운트 뱃지입니다.

## 언제 사용하나요?

- 제목/라벨 옆에 처리 건수나 항목 수를 표시할 때
- 대기·완료 같은 상태를 색으로 강조하고 싶을 때
- 값이 0이면 자동으로 숨기고 싶을 때

## Props

<Controls />

| Prop           | Description                                                            |
| -------------- | --------------------------------------------------------------------- |
| `count`        | 표시할 개수입니다.                                                    |
| `tone`         | 시각적 의미입니다. `neutral` · `danger` · `success`.                  |
| `hideWhenZero` | `true`(기본)면 `count` 가 0 이하일 때 렌더하지 않습니다.              |

## Behavior

- `hideWhenZero=true` 이고 `count <= 0` 이면 아무것도 렌더하지 않습니다.
- `hideWhenZero=false` 면 0 이나 음수도 그대로 표시합니다.
- 자릿수가 늘어도 pill 형태가 깨지지 않습니다.

## Accessibility

- 숫자만으로는 의미가 없으므로 인접한 제목이 문맥을 제공해야 합니다.
- 비대화형이라 포커스 순서에 들어가지 않습니다.
