# Button

사용자의 액션을 실행하는 기본 버튼 컴포넌트입니다.

## 언제 사용하나요?

- 저장·삭제·제출 같은 주요 액션을 실행할 때
- 취소·보조 액션을 primary 옆에 나란히 둘 때
- 위계에 따라 variant(primary·secondary·ghost·danger)를 나눌 때
- 처리 중임을 스피너로 알려야 할 때

## Props

<Controls />

| Prop          | Description                                                        |
| ------------- | ----------------------------------------------------------------- |
| `variant`     | 시각 위계입니다. `primary`·`secondary`·`ghost`·`danger` 중 하나.  |
| `size`        | 버튼 크기입니다. `sm`·`md`·`lg`.                                  |
| `loading`     | 처리 중 스피너를 표시하고 클릭을 막습니다.                        |
| `disabled`    | 버튼을 비활성화하고 클릭을 막습니다.                              |
| `iconLeft`    | 레이블 왼쪽에 표시할 아이콘입니다. 로딩 중에는 숨겨집니다.        |
| `isFullWidth` | 컨테이너 폭을 100%로 채웁니다.                                    |

## Behavior

- `loading` 또는 `disabled` 상태에서는 `onClick` 이 발화하지 않습니다.
- `type` 기본값은 `button` 입니다. 폼 안에서 실수로 제출되지 않도록 HTML 기본값(`submit`)을 뒤집었습니다.
- 폼을 제출하는 버튼에는 `type="submit"` 을 명시하세요.

## Accessibility

- 키보드로 포커스를 받고 `Enter`·`Space` 로 눌립니다.
- `disabled` 일 때 `aria-disabled`, `loading` 일 때 `aria-busy` 가 설정됩니다.
- `aria-label`·`title` 같은 표준 HTML/ARIA 속성을 그대로 전달합니다.
