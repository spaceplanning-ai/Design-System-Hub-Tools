# Checkbox

라벨이 붙은 체크박스입니다. 켜짐/꺼짐 상태를 사용자가 직접 토글합니다.

## 언제 사용하나요?

- 하나의 선택을 켜고 끌 때 (예: "로그인 상태 유지")
- 약관·동의처럼 명시적 확인이 필요할 때
- 라벨 클릭만으로도 토글되게 하고 싶을 때

## Props

<Controls />

| Prop       | Description                                                       |
| ---------- | ---------------------------------------------------------------- |
| `id`       | input 의 id. `label` 을 연결하는 기준입니다. (필수)              |
| `label`    | 가시 라벨. 이 텍스트를 클릭해도 토글됩니다. (필수)               |
| `checked`  | 체크 상태. 제어 값이라 `onChange` 로만 바뀝니다. (필수)          |
| `onChange` | 상태가 바뀔 때 호출됩니다. `disabled` 면 발화하지 않습니다.       |
| `name`     | 폼 제출 키. 빈 문자열이면 속성을 부여하지 않습니다.              |
| `disabled` | 비활성 상태. 토글되지 않습니다.                                  |

## Behavior

- 제어 컴포넌트입니다. `checked` 와 `onChange` 를 항상 짝지어 넘겨야 합니다. (비제어 `defaultChecked` 미지원)
- `disabled` 면 컨트롤·라벨 어느 쪽을 눌러도 `onChange` 가 발화하지 않습니다.

## Accessibility

- `Space` 로 토글, `Tab`·`Shift+Tab` 으로 이동합니다.
- 라벨은 `htmlFor={id}` 로 연결되어 라벨 클릭이 히트 영역이 됩니다.
- 체크 상태는 native 체크박스가 스스로 노출하므로 별도 aria 속성이 필요 없습니다.
- 부분 선택(indeterminate)은 지원하지 않습니다.
