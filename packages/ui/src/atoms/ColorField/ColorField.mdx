# ColorField

색 스와치와 hex 텍스트로 같은 색 값을 함께 보고 편집하는 입력 컴포넌트입니다.

## 언제 사용하나요?

- 배경·테두리·글자 색 등 색상을 고를 때
- 디자이너가 준 hex 코드를 직접 받아 적어야 할 때
- 색을 눈으로 고르는 방식과 코드로 입력하는 방식을 함께 제공하고 싶을 때

## Props

<Controls />

| Prop       | Description                                                          |
| ---------- | ------------------------------------------------------------------- |
| `value`    | 현재 색. `#RGB` · `#RRGGBB` · `#RRGGBBAA` 를 받습니다.                |
| `label`    | 스크린 리더용 이름. 스와치와 텍스트 입력을 구분해 안내합니다.        |
| `id`       | hex 텍스트 입력의 DOM id. 비우면 렌더하지 않습니다.                  |
| `disabled` | 스와치와 텍스트를 함께 잠급니다.                                     |
| `onChange` | 값이 유효한 hex 로 확정될 때 호출됩니다.                             |

## Behavior

- 스와치와 hex 텍스트 중 어느 쪽을 바꿔도 같은 `onChange` 로 나갑니다.
- 입력 도중 부분 문자열(`#6B`)은 올리지 않고, 유효한 hex 가 됐을 때만 `onChange` 를 발화합니다.
- 유효하지 않은 채 포커스를 떠나면 마지막 유효값으로 되돌립니다.
- `disabled` 면 두 입력 모두 잠기고 `onChange` 가 발화하지 않습니다.

## Accessibility

- 스와치와 텍스트 입력에 각각 접근 가능한 이름이 제공됩니다.
- `Tab` 으로 이동하고 `Enter` · `Space` 로 조작합니다.
- 포커스 표시(focus-visible)를 지원합니다.
- 논리 속성을 사용해 RTL 문서 방향을 따릅니다.
