# Divider

내용을 시각적으로 가르는 얇은 선입니다.

## 언제 사용하나요?

- 목록이나 섹션 사이를 시각적으로 구분할 때
- 툴바에서 액션 묶음 사이를 가를 때
- 세로선으로 나란한 항목을 나눌 때

<Controls />

`orientation` 을 `vertical` 로 두면 부모(flex 행) 높이에 맞춰 늘어납니다.

## Accessibility

- 장식 요소라 접근성 트리에서 숨겨집니다.
- 비대화형이라 포커스 순서에 들어가지 않습니다.

## Notes

- 여백은 부모가 줍니다. Divider 자체는 `margin` 을 갖지 않습니다.
