# HelpTip

라벨 옆에 붙어 눌러서 여닫는 ⓘ 도움말입니다.

## 언제 사용하나요?

- 폼 라벨에 부가 설명을 덧붙일 때
- 짧은 안내를 필요할 때만 펼쳐 보이고 싶을 때
- hover 툴팁 대신 키보드·터치에서도 열리는 도움말이 필요할 때

## Props

<Controls />

| Prop       | Description                                                     |
| ---------- | -------------------------------------------------------------- |
| `label`    | 트리거의 접근 가능한 이름입니다. 무엇에 대한 도움말인지 밝힙니다. |
| `children` | 열렸을 때 아이콘 아래로 뜨는 설명 문단의 내용입니다.            |

## Behavior

- 트리거(ⓘ)를 누르면 설명 패널이 열리고, 다시 누르면 닫힙니다.
- 열림 상태는 컴포넌트 내부가 관리합니다 — 외부에서 제어하지 않습니다.

## Accessibility

- 트리거는 버튼이며 `Enter`·`Space`·`Tab` 으로 동작합니다.
- 아이콘만 있으므로 `label` 이 접근 가능한 이름이 됩니다.
- `aria-expanded` 로 열림 상태를, `aria-controls` 로 설명 문단을 잇습니다.
- 닫혀도 패널을 언마운트하지 않고 `hidden` 으로 감춥니다.
