# Icon

`name` 으로 골라 쓰는 단색 선 아이콘입니다.

## 언제 사용하나요?

- 버튼·메뉴·목록 옆에 의미를 보조하는 그림이 필요할 때
- 텍스트와 함께 상태나 동작을 시각적으로 나타낼 때
- 색은 부모를 따르고 크기만 맞추면 될 때

## Props

<Controls />

| Prop    | Description                                                              |
| ------- | ----------------------------------------------------------------------- |
| `name`  | 그릴 아이콘. 사용 가능한 값은 위 Gallery 에서 확인하세요.                |
| `size`  | 크기. 기본값 `inherit` 는 부모 글자 크기를 따르고, `sm`·`md`·`lg` 로 고정할 수 있습니다. |
| `label` | 있으면 의미 아이콘, 없으면 장식으로 처리됩니다.                          |

색은 항상 `currentColor` 로 부모를 따릅니다.

## Accessibility

- `label` 이 있으면 `role=img` 로 이름을 읽어 줍니다.
- `label` 이 없으면 장식으로 보고 화면 낭독기에서 숨깁니다.
- 방향 아이콘도 RTL 에서 미러링하지 않습니다.
