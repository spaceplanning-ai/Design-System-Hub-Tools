# IconButton

아이콘만 있는 정사각 버튼입니다. 접근 가능한 이름(`label`)이 반드시 필요합니다.

## 언제 사용하나요?

- 편집기·툴바에서 아이콘만으로 액션을 제공할 때
- 되돌리기·다시하기 같은 단일 액션 버튼이 필요할 때
- 패널 접기·뷰 전환처럼 눌림 상태가 있는 토글이 필요할 때

## Props

<Controls />

| Prop       | Description                                                          |
| ---------- | ------------------------------------------------------------------- |
| `icon`     | 버튼에 표시할 아이콘입니다. (필수)                                   |
| `label`    | 접근 가능한 이름입니다. `aria-label` 과 `title` 에 함께 쓰입니다. (필수) |
| `size`     | 버튼 크기입니다. `md`(기본) · `sm`.                                 |
| `pressed`  | 토글 상태입니다. `unset` 이면 일반 액션, `on`/`off` 면 토글 버튼입니다. |
| `disabled` | 비활성화합니다. 클릭이 막히고 포커스 순서에서 제외됩니다.            |

## Behavior

- `pressed="unset"` 은 일반 액션(되돌리기 등), `on`/`off` 는 토글 버튼입니다. 일반 액션에 `off` 를 쓰면 스크린리더가 잘못 읽으니 주의하세요.
- `disabled` 는 클릭을 막습니다.

## Accessibility

- `label` 은 필수입니다. 비면 스크린리더가 "버튼" 으로만 읽습니다.
- 키보드 `Tab` 으로 포커스, `Enter`/`Space` 로 활성화됩니다.
- 키보드 포커스에만 포커스 링이 표시됩니다.
- `aria-expanded`·`aria-haspopup`·`aria-controls` 등은 `<button>` 으로 그대로 전달됩니다.
