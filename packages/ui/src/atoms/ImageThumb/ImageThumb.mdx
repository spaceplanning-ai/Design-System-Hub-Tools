# ImageThumb

목록에 쓰는 작은 썸네일 이미지입니다. 비어 있거나 로드에 실패하면 대체 이미지(placeholder)를 대신 보여줍니다.

## 언제 사용하나요?

- 목록에서 로고·인증서·리뷰 등 작은 미리보기 이미지를 보여줄 때
- 이미지 URL 이 없거나 깨질 수 있어 대체 표시가 필요할 때

## Props

<Controls />

| Prop  | Description                                                                    |
| ----- | ----------------------------------------------------------------------------- |
| `src` | 이미지 URL 입니다. 앞뒤 공백을 뺀 뒤 빈 문자열이면 placeholder 를 보여줍니다. |
| `alt` | 접근성 이름입니다. 실제 이미지와 placeholder 모두 같은 이름으로 읽힙니다.     |

## Behavior

- `src` 로드에 실패하면 깨진 이미지 대신 placeholder 로 대체합니다.
- `src` 가 바뀌면 실패 상태를 초기화합니다 — 이전 URL 의 실패가 새 URL 로 이어지지 않습니다.

## Accessibility

- 실제 이미지와 placeholder 모두 `alt` 로 같은 접근 가능한 이름을 제공합니다.
- 비대화형 표시 전용이라 포커스 순서에 들어가지 않습니다.
