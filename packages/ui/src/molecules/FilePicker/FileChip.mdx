# FileChip

<code>molecule</code> · <code>File</code> · <code>taxonomyItem: file-preview</code> · <code>v1.0.0</code> · <code>status: beta</code>

지금 걸려 있는 파일 한 건 — 썸네일 + 파일명 + 용량, 그리고 **선택적** 제거(×).
출처: `apps/admin/src/shared/ui/FilePicker.tsx` (사이트 설정의 파비콘·대표 이미지·비공개 이미지가 공유).

### FileDropzone 과 짝이다

계약 `description`: 둘은 가로로 나란히 서고, 미리보기는 드롭존이 아니라 **브라우저 탭 목업·OG 카드라는
다른 자리**에 그려진다. 그래서 미리보기를 자기가 소유하는 ImageUploadField 로 대체할 수 없다 —
**미리보기의 주인이 다르다.**

## Props

<Controls />

| prop | 타입 | 기본값 | 의미 |
|---|---|---|---|
| `src` | `string` | `''` | 썸네일 URL. 비거나 로드 실패면 **ImageThumb 가 placeholder 를 그린다.** |
| `name` | `string` | — (필수) | 파일명 — 한 줄로 자르고 넘치면 말줄임한다. **제거 버튼의 접근 가능한 이름(`'<name> 제거'`)도 여기서 온다.** |
| `size` | `number` | — (필수) | 바이트 수 — 표기는 `formatFileSize` 가 한다. **음수/NaN 은 `'-'`.** |
| `disabled` | `boolean` | `false` | 비활성 — 제거 버튼을 잠근다(**칩 자체는 계속 읽힌다**). `onRemove` 발화 없음. |

### 용량 표기 규칙

계약 `description`: `formatFileSize` **순수 함수**가 소유한다. 1KB 미만 B · 1MB 미만 KB · 그 위 MB 소수 1자리.

**소수점은 MB 부터만 붙인다** — KB 단위의 '12.7KB' 는 정보가 아니라 소음이고,
칩이 보여줄 것은 '작다/크다' 이지 정확한 바이트가 아니다.

## 접근성 계약

- **role**: `group`
- **keyboard**: `Tab` · `Enter` · `Space`
- **focusVisible**: `true`
- **contrastMin**: 4.5

계약 `a11y.aria`:

- `label`: 제거 버튼은 `'<name> 제거'` 를 `aria-label` 로 갖는다 —
  **칩이 여럿 서 있어도 어느 파일을 지우는지 알 수 있다.**
- `thumbnail`: 썸네일 `alt` 는 `'<name> 썸네일'` — ImageThumb 가 로드 실패 시 `role=img` placeholder 로 대체한다.
- `locking`: `disabled` 이면 제거 `<button disabled>` 로 잠근다.
