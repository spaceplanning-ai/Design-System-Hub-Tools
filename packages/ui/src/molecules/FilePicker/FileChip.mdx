# FileChip

첨부 파일 한 건을 이름·크기와 함께 표시하고, 제거 버튼을 제공하는 컴포넌트입니다.

## 언제 사용하나요?

- 이미 첨부된 파일 한 건을 이름·용량과 함께 보여줄 때
- 파일을 지울 수 있는 제거(×) 버튼이 필요할 때
- 파일 선택/업로드 UI 옆에 현재 상태를 표시할 때

## Props

<Controls />

| Prop       | Description                                                        |
| ---------- | ----------------------------------------------------------------- |
| `src`      | 썸네일 URL. 비거나 로드에 실패하면 placeholder 를 대신 보여줍니다. |
| `name`     | 파일명. 한 줄로 자르고 넘치면 말줄임합니다.                        |
| `size`     | 파일 크기(바이트). 사람이 읽는 표기로 바뀝니다.                    |
| `disabled` | 제거 버튼을 잠급니다. 칩 자체는 계속 표시됩니다.                   |
| `onRemove` | 제공하면 제거(×) 버튼이 표시되고, 누르면 호출됩니다.               |

## Behavior

- `onRemove` 를 주지 않으면 제거(×) 버튼이 아예 표시되지 않습니다.
- `disabled` 면 제거 버튼이 잠겨 `onRemove` 가 호출되지 않습니다.
- 크기 표기: 1KB 미만은 B, 1MB 미만은 KB, 그 이상은 MB(소수 1자리). 음수·NaN 은 `-`.

## Accessibility

- 제거 버튼에는 `'<파일명> 제거'` 접근 이름이 제공되어, 칩이 여럿이어도 어느 파일을 지우는지 알 수 있습니다.
- 썸네일에는 `'<파일명> 썸네일'` 대체 텍스트가 제공됩니다.
- 논리 속성을 사용해 RTL 환경에서 자동으로 방향이 뒤집힙니다.
