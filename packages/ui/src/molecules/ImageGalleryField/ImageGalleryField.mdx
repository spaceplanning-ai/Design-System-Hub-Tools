# ImageGalleryField

여러 이미지를 업로드하고 미리보기·제거할 수 있는 갤러리 입력 필드입니다.

## 언제 사용하나요?

- 한 필드에서 여러 장의 이미지를 등록받을 때
- 등록한 이미지를 그리드로 미리보고 개별로 지우게 할 때
- 파일 형식·용량·개수에 상한을 두고 싶을 때

## Props

<Controls />

| Prop        | Description                                                       |
| ----------- | ---------------------------------------------------------------- |
| `label`     | 필드 라벨입니다. 드롭존 이름과 각 타일 alt 의 기준이 됩니다.      |
| `values`    | 현재 이미지 URL 배열입니다. 비어 있으면 드롭존만 표시됩니다.      |
| `onChange`  | 이미지가 추가·제거될 때 새 URL 배열로 호출됩니다.                 |
| `required`  | 라벨에 필수 표식(\*)을 붙입니다.                                  |
| `disabled`  | 추가·제거를 모두 비활성화합니다.                                  |
| `error`     | 지정하면 오류 테두리와 인라인 오류 메시지를 표시합니다.          |
| `hint`      | 도움말 문구입니다. 오류가 없을 때만 표시됩니다.                   |
| `maxFiles`  | 등록 가능한 최대 장수입니다. (기본 10)                           |
| `maxSizeMB` | 파일당 용량 상한(MB)입니다. (기본 5)                             |

## Behavior

- **추가**: 드롭존을 클릭하거나 이미지를 끌어다 놓으면 파일을 고를 수 있습니다.
- **제거**: 각 타일의 제거 버튼으로 해당 이미지만 뺍니다.
- **최대 개수**: `maxFiles` 만큼 차면 '추가' 칸이 사라집니다.
- **검증**: `image/*` 가 아니거나 용량·개수를 넘으면 인라인 오류로 막습니다.

## Accessibility

- 드롭존과 '추가' 칸은 버튼이며 `Enter`·`Space`·`Tab` 으로 조작할 수 있습니다.
- 각 타일 제거 버튼에는 '`N`번째 이미지 제거' 라는 접근 가능한 이름이 있습니다.
- 실제 파일 입력은 시각적으로 숨겨져 탭 순서 밖에 있습니다.
- 오류는 즉시 통지되며 드롭존과 연결됩니다.
