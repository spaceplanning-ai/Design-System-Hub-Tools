# ImageUploadField

이미지 1장을 업로드하고 미리보기를 보여주는 필드입니다.

## 언제 사용하나요?

- 로고·대표 사진처럼 이미지 1장을 받을 때
- 클릭 또는 드래그드롭으로 파일을 선택하게 할 때
- 선택한 이미지를 미리보기로 확인·교체·제거해야 할 때
- 폼에서 이미지 필드에 라벨·도움말·오류를 함께 보여줄 때

## Props

<Controls />

| Prop        | Description                                                       |
| ----------- | ---------------------------------------------------------------- |
| `label`     | 필드 라벨. 드롭존의 접근 가능한 이름과 미리보기 `alt` 의 기준입니다. |
| `value`     | 현재 이미지 URL. 빈 문자열이면 미등록 상태입니다.                 |
| `onChange`  | 값이 바뀔 때 새 URL 문자열을 넘깁니다. 제거하면 빈 문자열을 넘깁니다. |
| `required`  | 라벨에 필수 표식(\*)을 붙입니다.                                  |
| `disabled`  | 선택·교체·제거를 모두 비활성화합니다.                            |
| `error`     | 표시할 오류 문구. 있으면 danger 테두리와 인라인 오류를 그립니다.  |
| `hint`      | 도움말 문구. 오류가 없을 때만 표시됩니다.                         |
| `maxSizeMB` | 허용 파일 용량 상한(MB). 초과하면 오류로 막습니다. 기본값 5.      |

## Behavior

- **선택**: 클릭하거나 파일을 드래그해 놓으면 이미지를 올립니다.
- **제거**: '제거' 를 누르면 값이 빈 문자열이 되고 미리보기가 사라집니다.
- **검증**: 이미지 타입이 아니거나 `maxSizeMB` 를 넘으면 인라인 오류로 막습니다.
- 값이 있으면 미리보기와 함께 '이미지 교체' · '제거' 버튼이 나타납니다.

## Accessibility

- 드롭존은 `<button>` 이라 `Tab` 으로 포커스가 닿고 `Enter` · `Space` 로 파일 선택을 엽니다.
- 라벨은 드롭존의 접근 가능한 이름으로 이어지고, 필수일 때 이름에 '(필수)' 가 더해집니다.
- 오류는 `role=alert` 로 즉시 안내되고 `aria-describedby` 로 드롭존에 연결됩니다.

## Notes

- 지금 `onChange` 값은 `URL.createObjectURL` 미리보기 핸들입니다 — 저장된 자산이 아닙니다.
  백엔드가 붙으면 업로드 응답 URL 로 대체됩니다. 이 값을 영속 저장하면 썸네일이 깨집니다.
