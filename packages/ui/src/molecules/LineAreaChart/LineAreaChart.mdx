# LineAreaChart

<code>molecule</code> · <code>Charts</code> · <code>taxonomyItem: line-chart</code> · <code>v1.1.0</code> · <code>status: beta</code>

범용 선 + 면적 차트 — 계열마다 `kind` 로 선/면적을 고른다.

**외부 차트 라이브러리 의존 0** (SVG 좌표 직접 계산, Catmull-Rom → 3차 베지어 스무딩).
**도메인 중립**: '방문자/페이지뷰' 같은 계열 의미는 데이터를 주입하는 organism 이 소유한다 (ADR-0003).
출처 구현: `apps/admin/src/pages/dashboard/components/VisitorChart.tsx` 를 일반화.

## Props

| prop | 타입 | 기본값 | 의미 |
|---|---|---|---|
| `series` | `array` | — (필수) | 계열 목록. **`values` 길이는 `labels` 길이와 같아야 한다.** 데이터 prop — Figma 대응 없음 (ADR-0003). |
| `labels` | `array` | — (필수) | x축 눈금 라벨. 데이터 prop — Figma 대응 없음. |
| `showLegend` | `boolean` | `true` | 계열 범례 표시 여부. **범례는 장식이 아니라 계열 식별 수단이므로 기본 노출.** |
| `ariaLabel` | `string` | — (필수) | `role=img` 의 접근 가능한 이름. **차트가 전달하는 추세를 문장으로 기술한다.** |

**이벤트가 없다** — 계약에 `events` 항목 자체가 없다.

## States

계약 `states` 는 `default` 하나뿐이다.

> `Loading` 스토리가 존재하지만 **계약의 `states` 에는 loading 이 없다.** StatsCard 계약이 기록한
> '진입 불가 loading 상태 제거' 판정과 같은 계열의 사안이다 — 이 문서는 계약을 옮길 뿐 판정하지 않는다.

## 접근성 계약

- **role**: `img`
- **keyboard**: `Tab`
- **focusVisible**: `true`
- **contrastMin**: **3** — 다른 대부분의 컴포넌트가 4.5 인 것과 다르다. 그래픽 객체 기준이다.

계약 `a11y.aria`:

- `aria-label`: `ariaLabel` prop — 차트 전체의 이름.
- `aria-hidden`: 범례 색 점 — **장식용이므로 `true`.**

## 토큰

차트 축·라벨: `color.chart.axis`(gridLine) · `color.chart.label`(axisLabel·legendText) ·
`radius.full`(legendDotRadius) · `space.4`(legendGap) · `space.1`(legendItemGap).

계열 색은 **6계열까지 토큰으로 고정된다**:

| 2 | `color.chart.series-2` | `color.chart.series-2-fill` |
| 3 | `color.chart.series-3` | `color.chart.series-3-fill` |
| 4 | `color.chart.series-4` | `color.chart.series-4-fill` |
| 5 | `color.chart.series-5` | `color.chart.series-5-fill` |
| 6 | `color.chart.series-6` | `color.chart.series-6-fill` |

그 외: `color.surface.default` · `typography.label.md` · `color.border.focus`(focusRing).
