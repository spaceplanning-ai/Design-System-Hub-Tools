# LineAreaChart

여러 계열의 추이를 선·영역으로 그리는 차트입니다.

## 언제 사용하나요?

- 시간에 따른 값의 추이를 보여줄 때
- 여러 계열을 한 차트에서 비교할 때
- 계열마다 선(line) 또는 면적(area)으로 다르게 표현하고 싶을 때

## Props

<Controls />

| Prop         | Description                                                                 |
| ------------ | -------------------------------------------------------------------------- |
| `series`     | 계열 목록입니다. 각 계열의 `values` 길이는 `labels` 길이와 같아야 합니다.   |
| `labels`     | x축 눈금 라벨입니다.                                                        |
| `showLegend` | 계열 범례 표시 여부입니다. 기본값은 `true` 입니다.                          |
| `ariaLabel`  | 차트의 접근 가능한 이름입니다. 차트가 전달하는 추세를 문장으로 적습니다.    |

각 계열의 `kind` 로 표현을 고릅니다. `area` 는 곡선 아래를 채운 면적, `line` 은 선 + 점만 그립니다. 계열 색은 6계열까지 자동으로 서로 다르게 지정됩니다.

## Accessibility

- 차트 전체가 `role=img` 로 `ariaLabel` 을 접근 가능한 이름으로 갖습니다.
- 범례의 색 점은 장식이므로 스크린리더에서 감춰집니다.
- `ariaLabel` 에는 추세를 요약한 문장을 넣어 주세요.
