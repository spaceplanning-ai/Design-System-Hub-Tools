# ListRow

목록 한 줄 — 클릭 또는 링크 행. 좌측 아이콘 + 제목 + 메타(보조 텍스트)로 구성됩니다.

## 언제 사용하나요?

- 목록에서 항목 하나를 눌러 상세로 이동시킬 때
- 클릭하면 동작이 실행되는 행이 필요할 때
- 제목 아래 작성자·날짜 같은 보조 정보를 함께 보여줄 때

## Props

<Controls />

| Prop    | Description                                                          |
| ------- | ------------------------------------------------------------------- |
| `title` | 행 제목입니다. 길면 줄바꿈됩니다. (필수)                            |
| `meta`  | 제목 아래 보조 텍스트입니다. 비어 있으면 표시하지 않습니다.         |
| `icon`  | 좌측 아이콘 슬롯입니다. 장식용으로만 쓰입니다.                       |
| `href`  | 링크 대상입니다. 값이 있으면 링크 행, 없으면 클릭 행으로 렌더합니다. |

## Behavior

- `href` 가 있으면 `<a>` 링크로, 없으면 `<button>` 행으로 렌더합니다.
- 행을 누르면 `onClick` 이 호출됩니다. `href` 가 있어도 라우팅 가로채기·계측을 위해 함께 발화합니다.
- 목록 컨테이너(`ul`/`li`)는 조립하는 상위 컴포넌트(ListCard 등)가 소유합니다.

## Accessibility

- `Tab` 으로 행에 포커스가 들어오고, `Enter` 로 실행됩니다.
- 포커스 시 포커스 링이 표시됩니다.
- 아이콘 슬롯은 장식용이라 보조 기술에서 감춰집니다.
