# Menu

트리거를 눌러 여는 드롭다운 명령 메뉴입니다.

## 언제 사용하나요?

- 표 행이나 툴바에서 '더보기' 액션을 모아 보여줄 때
- 값을 고르는 게 아니라 명령(삭제·내보내기 등)을 실행할 때
- 공간이 좁아 버튼을 나열하기 어려울 때

## Props

<Controls />

| Prop           | Description                                                     |
| -------------- | -------------------------------------------------------------- |
| `label`        | 트리거와 팝업의 접근 가능한 이름입니다. 행마다 달라야 합니다.   |
| `items`        | 명령 목록입니다. `{ id, label, danger?, disabled?, disabledReason? }` 모양입니다. |
| `align`        | 팝업이 붙는 모서리(`'start' \| 'end'`)입니다. RTL 에서 자동으로 뒤집힙니다. |
| `trigger`      | 트리거 글리프(`'more-horizontal' \| 'more-vertical'`)입니다.    |
| `onSelect`     | 항목을 선택하면 그 `id` 로 호출됩니다.                          |
| `onOpenChange` | 팝업이 열리거나 닫힐 때 호출됩니다.                             |

`disabledReason` 을 주면 `disabled` 를 따로 주지 않아도 그 항목은 잠기고, 이유가 화면에 표시됩니다.

## Behavior

- 트리거를 누르면 팝업이 열리고, 다시 누르거나 바깥을 클릭하면 닫힙니다.
- 항목을 선택하면 `onSelect(id)` 가 발화하고 팝업이 닫힙니다.
- `Escape` 로 닫으면 트리거로 포커스가 돌아오지만, 명령 실행으로 닫힐 때는 돌아오지 않습니다.
- 잠긴 항목은 목록에서 지우지 않고 잠근 채 이유를 함께 보여줍니다.

## Accessibility

- 트리거는 `Enter` · `Space` · `ArrowDown` 으로 열립니다.
- 팝업 안에서는 `ArrowUp` · `ArrowDown` 으로 항목 사이를 순환 이동하고, `Home` · `End` 로 처음·끝으로 갑니다.
- `Escape` 는 팝업을 닫고 트리거로 포커스를 되돌립니다.
- 열리면 첫 항목으로 포커스가 이동하며, 포커스를 가두지 않고 `Tab` 하면 닫습니다.
- `danger` 항목은 색만으로 의미를 전달하지 않도록 라벨 문구로도 위험을 알립니다.
