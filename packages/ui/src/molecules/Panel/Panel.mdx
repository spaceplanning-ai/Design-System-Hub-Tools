# Panel

본문 옆에 내용을 담는 구획 컨테이너입니다. 블록을 세로로 쌓고, 선택적으로 맨 아래에 안내 영역을 둡니다.

## 언제 사용하나요?

- 화면 곁에 필터·분류·내비게이션 같은 축을 세로로 쌓을 때
- 블록이 여럿이라 블록 사이 간격을 일정하게 맞춰야 할 때
- 맨 아래에 구분선으로 갈린 안내문을 붙이고 싶을 때

## Props

<Controls />

| Prop       | Description                                                            |
| ---------- | --------------------------------------------------------------------- |
| `children` | 패널이 세로로 쌓을 블록들. 블록 사이 간격은 패널이 맞춥니다. (필수)    |
| `notice`   | 맨 아래 안내 영역. 비우면 구분선과 여백까지 함께 사라집니다.           |

이벤트는 없습니다 — 상호작용은 전부 `children` 의 것입니다.

## Accessibility

<Story of={PanelStories.AriaRegion} />

- 패널은 `complementary` 랜드마크로 안내됩니다.
- 패널 자신은 이름·제목을 갖지 않습니다 — 이름은 안에 들어오는 각 블록(`<nav>` 등)이 갖습니다.
- 논리 속성만 쓰므로 RTL 에서 여백이 반대편으로 따라갑니다.

## Notes

- 패널은 자기 폭을 정하지 않습니다. 폭은 그 화면의 격자가 정합니다.
