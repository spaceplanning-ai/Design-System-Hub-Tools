# RadioCardGroup

카드 형태의 단일 선택 라디오 그룹입니다. 선택지마다 제목과 결과 설명을 함께 보여줍니다.

## 언제 사용하나요?

- 선택지가 적고 각각의 결과가 크게 다를 때
- 고르기 전에 각 선택지가 어떤 결과인지 읽혀야 할 때
- 접어 두는 드롭다운(SelectField)이 아니라 펼쳐서 비교하게 하고 싶을 때

## Props

<Controls />

| Prop       | Description                                                              |
| ---------- | ----------------------------------------------------------------------- |
| `name`     | 라디오들이 공유하는 이름입니다. 화면에 그룹이 둘 이상이면 서로 달라야 합니다. |
| `legend`   | 그룹의 보이는 제목이자 접근성 이름입니다.                                |
| `value`    | 선택된 항목의 value 입니다. `options[].value` 중 하나여야 합니다.        |
| `options`  | 선택지 목록입니다. 각 항목의 `description` 은 고르면 무슨 일이 일어나는지 설명합니다. |
| `disabled` | 그룹 전체를 비활성화하고 라벨을 흐리게 표시합니다.                       |
| `onChange` | 다른 카드를 고르면 그 항목의 value 로 호출됩니다.                        |

## Behavior

- 카드를 고르면 테두리·배경·라디오 점이 함께 바뀝니다.
- `disabled` 이면 모든 카드가 잠기고 `onChange` 가 호출되지 않습니다.

## Accessibility

- `role="radiogroup"` 이며 `legend` 로 그룹 이름을 갖습니다.
- 네이티브 `<input type="radio">` 라 화살표 키 이동과 단일 선택을 브라우저가 처리합니다.
- 각 카드의 설명은 `aria-describedby` 로 이어져 이름과 분리해 읽힙니다.
