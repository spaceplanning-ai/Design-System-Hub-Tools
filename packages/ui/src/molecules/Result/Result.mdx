# Result

전체 화면 결과·상태 안내 — 완료·오류·권한 없음 등 화면을 그릴 수 없을 때 그 자리를 대신 채우는 상태 블록입니다.

## 언제 사용하나요?

- 화면 전체가 실패했을 때 (렌더 예외·점검·정지)
- 접근 권한이 없을 때 (403)
- 요청한 페이지를 찾을 수 없을 때 (404)
- 사용자에게 빠져나갈 액션을 함께 안내할 때

## Props

<Controls />

| Prop          | Description                                                            |
| ------------- | --------------------------------------------------------------------- |
| `title`       | 무슨 일이 일어났는지 한 줄로. `h2` 로 렌더됩니다. (필수)               |
| `description` | 제목을 보충하는 한 문단. 빈 문자열이면 그리지 않습니다.               |
| `reference`   | 신고에 붙일 짧은 참조 코드. 빈 문자열이면 그리지 않습니다.            |
| `actions`     | 빠져나갈 수단. `<Button>` 하나 또는 여럿. 주지 않으면 액션 줄을 그리지 않습니다. |

## Behavior

- `description` · `reference` 는 빈 문자열이면 그 줄 자체를 그리지 않습니다 (빈 줄을 남기지 않음).
- `actions` 를 주지 않으면 제목만 남습니다.
- 참조 코드는 운영자가 그대로 복사해 티켓에 붙일 수 있게 한 번에 통째로 선택됩니다.

## Accessibility

- 화면이 그려지지 못했다는 사실은 즉시 안내됩니다 (`role="alert"`).
- 제목은 `h2` 로, 참조 코드까지 보조기술에 함께 전달됩니다.
- `dir="rtl"` 을 지원합니다.
