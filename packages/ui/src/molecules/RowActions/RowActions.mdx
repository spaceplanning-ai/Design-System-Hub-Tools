# RowActions

표·목록 행의 인라인 액션 버튼입니다 — 수정 · 삭제.

## 언제 사용하나요?

- 목록 행 끝에 수정/삭제 액션을 붙일 때
- 수정만, 또는 삭제만 필요할 때(핸들러 유무로 결정)
- 콘텐츠 목록(공지·FAQ·팝업·배너)의 행 액션을 한 벌로 맞출 때

## Props

<Controls />

| Prop       | Description                                                          |
| ---------- | ------------------------------------------------------------------- |
| `label`    | 스크린 리더용 대상 이름 — 행마다 달라야 합니다('공지 제목').         |
| `disabled` | 진행 중일 때 두 버튼을 모두 잠급니다.                                |
| `onEdit`   | 제공하면 수정(연필) 버튼이 표시됩니다. 미지정이면 그리지 않습니다.   |
| `onDelete` | 제공하면 삭제(휴지통) 버튼이 표시됩니다. 미지정이면 그리지 않습니다. |

## Behavior

- 액션을 누르면 `onEdit` / `onDelete` 가 호출됩니다.
- `disabled` 이면 클릭해도 콜백이 발화하지 않습니다.
- 삭제는 이 컴포넌트가 직접 지우지 않습니다 — 호출부가 `onDelete` 안에서 확인 다이얼로그를 엽니다.

## Accessibility

- 각 버튼은 `'{label} 수정'` / `'{label} 삭제'` 로 접근 가능한 이름을 갖습니다(색만으로 의미 전달하지 않음).
- `Tab` 으로 액션 버튼에 포커스가 이동하고 `Enter` · `Space` 로 실행합니다.
- `dir="rtl"` 에서는 액션 정렬이 반대로 뒤집힙니다.
