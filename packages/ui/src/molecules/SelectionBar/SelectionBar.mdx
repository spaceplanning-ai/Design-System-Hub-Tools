# SelectionBar

여러 행을 선택했을 때 나타나는 일괄 작업 바입니다.

## 언제 사용하나요?

- 목록에서 여러 행을 체크박스로 골랐을 때
- 선택한 항목을 한 번에 삭제·ON/OFF 하는 등 일괄 작업이 필요할 때
- 선택 개수를 사용자에게 보여줄 때

## Props

<Controls />

| Prop       | Description                                                          |
| ---------- | ------------------------------------------------------------------- |
| `count`    | 선택된 행 수입니다. `0` 이면 바를 그리지 않습니다. 천 단위로 표기합니다. |
| `noun`     | 개수 단위 문구입니다('건'·'명' 등). 기본값은 '건'.                   |
| `children` | 일괄 작업 버튼들입니다(일괄 삭제·수정 등). 호출부가 버튼으로 넣습니다. |
| `onClear`  | 제공하면 '선택 해제' 버튼이 표시되고, 누르면 호출됩니다.             |

## Behavior

- `count` 가 `0` 이면 바 자체가 렌더되지 않습니다.
- `onClear` 를 주면 '선택 해제' 버튼이 그려지고, 미지정이면 그리지 않습니다.
- 일괄 작업은 `children` 으로 넣은 버튼의 `onClick` 이 담당합니다.

## Accessibility

- 선택 바는 접근성 이름('선택 항목 일괄 작업')을 가진 영역으로 안내됩니다.
- `Tab` 으로 '선택 해제' 버튼에 포커스를 이동할 수 있습니다.
- 파괴적 버튼도 색만이 아니라 '삭제' 같은 라벨로 의미를 전달합니다.
