# Stepper

다단계 진행 상태를 번호 점과 라벨로 표시하는 비대화형 표시기입니다.

## 언제 사용하나요?

- 정해진 흐름의 단계와 현재 위치를 보여줄 때
- 반품 처리·영업 파이프라인처럼 순서가 고정된 진행 과정을 나타낼 때
- 값을 입력·증감하지 않고 상태만 읽히면 될 때

## Props

<Controls />

| Prop        | Description                                             |
| ----------- | ------------------------------------------------------ |
| `steps`     | 단계 목록. 배열 순서가 곧 흐름의 순서입니다.           |
| `current`   | 현재 단계의 id. 이 단계까지 채워집니다.                |
| `ariaLabel` | 목록(`<ol>`)의 접근 가능한 이름입니다.                 |

- 이벤트가 없습니다. 단계를 눌러 흐름을 바꿀 수 없습니다.
- 목록에 없는 `current`(반려·실주 같은 흐름 밖 종료)가 오면 아무 단계도 채우지 않습니다. 그 사실은 호출부가 danger 배너·배지로 따로 알립니다.

## Accessibility

<Story of={StepperStories.NonInteractive} />
<Story of={StepperStories.RightToLeft} />

- 현재 단계의 `<li>` 에 `aria-current="step"` 이 붙어 진행 위치를 비시각 사용자에게 전합니다.
- 비대화형입니다. 포커스 순서에 들어가지 않고 Tab 으로 단계에 접근할 수 없습니다.
- 점 안의 번호와 연결선은 장식이라 `aria-hidden` 이고, 순서는 `<ol>` 이 전달합니다.
