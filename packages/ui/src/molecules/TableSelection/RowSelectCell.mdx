# RowSelectCell

<code>molecule</code> · <code>Tables</code> · <code>v1.0.0</code> · <code>status: beta</code>

표 행의 선택 칸 — **보이지 않는 라벨 + 단일 체크박스를 담은 `<td>`.**

목록 표(회원·운영자·공지·FAQ·팝업·배너·버전 이력)의 행 선택 마크업을 한 벌로 올렸다.
**도메인을 모른다** — 행 식별자(`id`)·보이지 않는 라벨 문구(`label`)·`checked`·토글 콜백만 받는다.
출처: `apps/admin/src/shared/ui/TableSelection.tsx` 의 `RowSelectCell`
(TableSelection 묶음의 **행-셀 대표**).

> 이 컴포넌트의 계약에는 `taxonomyItem` 이 없다.

## Props

<Controls />

| prop | 타입 | 기본값 | 의미 |
|---|---|---|---|
| `id` | `string` | — (필수) | 행 식별자 — 보이지 않는 라벨의 id(`'select-{id}'`)를 파생한다(**한 문서에 여러 표가 있어도 안 겹친다**). |
| `label` | `string` | — (필수) | 보이지 않는 라벨 문구 — 예: `'{공지 제목} 선택'`. `aria-labelledby` 로 체크박스에 이어진다. |
| `checked` | `boolean` | — (필수) | 선택 여부. 제어 값 — `onToggle` 로만 바뀐다. |

### 보이는 라벨이 없다

계약 `description`: 보이는 라벨이 없으므로 **시각적으로 숨긴 문구를 두고 `aria-labelledby`(id 파생)로 잇는다.**

## 접근성 계약

- **role**: `checkbox`
- **keyboard**: `Space` · `Tab`
- **focusVisible**: `true`
- **contrastMin**: 4.5

계약 `a11y.aria`:

- `aria-labelledby`: 시각적으로 숨긴 라벨(`span#select-{id}`)을 가리킨다 —
  **스크린리더가 '무엇을 고르는지' 읽는다.**

## 토큰

`color.action.primary.default`(control) · `color.text.default`(cellText) ·
`color.border.default`(cellBorder) · `space.2`(cellPaddingX) · `space.3`(cellPaddingY) ·
`space.4`(controlSize) · `space.6`(cellWidth) · `color.border.focus`(focusRing) · `border-width.thin`.

`responsive.behavior` 는 `size-fixed`. 의존 없음.
