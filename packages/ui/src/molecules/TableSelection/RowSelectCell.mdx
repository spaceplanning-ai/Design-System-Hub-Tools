# RowSelectCell

표 각 행의 선택 체크박스 셀입니다.

## 언제 사용하나요?

- 목록 표(회원·운영자·공지·FAQ·팝업·배너 등)에서 행 하나를 선택할 때
- 여러 행을 골라 일괄 작업(삭제·상태 변경 등)을 하려 할 때

## Props

<Controls />

| Prop       | Description                                                     |
| ---------- | -------------------------------------------------------------- |
| `id`       | 행 식별자입니다. 라벨 id(`select-{id}`)를 파생해 표끼리 겹치지 않게 합니다. |
| `label`    | 스크린리더가 읽을 라벨 문구입니다. 예: `'{공지 제목} 선택'`.        |
| `checked`  | 선택 여부입니다. `onToggle` 로만 바뀌는 제어 값입니다.            |
| `onToggle` | 선택 상태가 바뀔 때 다음 값(`boolean`)과 함께 호출됩니다.         |

## Accessibility

- 체크박스에는 보이는 라벨이 없어, 시각적으로 숨긴 문구를 `aria-labelledby` 로 이어 접근 이름을 제공합니다.
- `Space` 로 토글, `Tab` 으로 이동하며 포커스 링이 보입니다.
