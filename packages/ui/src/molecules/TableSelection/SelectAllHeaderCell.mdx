# SelectAllHeaderCell

표 헤더의 전체 선택 체크박스 — 부분 선택을 포함한 3상태(선택 없음·전체·부분).

## 언제 사용하나요?

- 표의 모든 행을 한 번에 선택·해제할 때
- 일부 행만 선택된 상태를 헤더에서 보여줘야 할 때
- 회원·운영자·콘텐츠 목록처럼 페이지네이션된 표의 헤더 칸이 필요할 때

## Props

<Controls />

| Prop         | Description                                                                              |
| ------------ | --------------------------------------------------------------------------------------- |
| `label`      | 화면에 보이지 않는 라벨 문구입니다. 예: `'이 페이지의 회원 전체 선택'`.                  |
| `labelId`    | 라벨 요소의 id 입니다. 한 문서에 표가 여러 개면 표마다 다른 값을 줍니다.                 |
| `selection`  | 이 페이지의 선택 상태입니다. `allSelected` 면 전체, `someSelected` 면 부분으로 표시됩니다. |
| `onToggleAll`| 체크박스를 누르면 다음 상태(`boolean`)와 함께 호출됩니다.                                |

동반 유틸 `tableSelectionState(rows, selectedIds)` 가 지금 보이는 행만 세어 `selection` 값을 만들어 줍니다. 다른 페이지의 선택은 세지 않습니다.

## Accessibility

- 전체 선택은 `aria-checked="true"`, 부분 선택은 `aria-checked="mixed"`, 선택 없음은 `false` 로 안내됩니다.
- 보이는 라벨이 없으므로 숨긴 라벨을 `aria-labelledby` 로 연결합니다.
- `Space` 로 토글, `Tab` 으로 이동할 수 있습니다.

## Notes

- 행 쪽 선택 칸은 **RowSelectCell** 이며, 그쪽은 단일 체크박스(2상태)를 씁니다.
