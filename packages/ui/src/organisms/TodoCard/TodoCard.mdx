# TodoCard

<code>organism</code> · <code>Data Display</code> · **<code>v2.0.0</code>** · <code>status: beta</code>

'오늘의 할일' 카드. **Card + Badge 를 조립해** 처리 대기 건수를 한 줄로 보여주는 업무 위젯
(ADR-0003 organism).

`items` 는 데이터 prop 이라 Figma 대응이 없다 — **탭(상품·문의·영업)에 따라 항목 집합이 통째로 바뀌므로**
항목을 계약에 고정하지 않고 호출부(Pages)가 주입한다.

> 이 컴포넌트의 계약에는 `taxonomyItem` 이 없다.

## Props

<Controls />

| prop | 타입 | 기본값 | 의미 |
|---|---|---|---|
| `title` | `string` | `'오늘의 할일'` | 카드 제목. |
| `items` | `array` | — (필수) | 할일 항목 목록. **`count > 0` 인 항목은 강조색(`color.feedback.danger.text`), `0` 인 항목은 흐리게(`color.text.disabled`)** 렌더한다. `href` 가 있으면 링크, 없으면 정적 텍스트. 데이터 prop — Figma 대응 없음 (ADR-0003). |
| `loading` | `boolean` | `false` | 로딩 중 스켈레톤 표시 + `aria-busy` + **`onItemClick` 차단.** |
| `showTotal` | `boolean` | `true` | 제목 옆에 `items` 의 count **합계**를 Badge 로 표시. |

## 2.0.0 — SPA 내비게이션 탈출구 (파괴적 변경)

계약 `description`: **ListCard 와 동일 결함·동일 처방이다** (같은 판정으로 함께 올린다).
`onItemClick` 이 `MouseEvent` 를 함께 넘겨 호출부가 `preventDefault()` 후 `navigate()` 할 수 있다.

| 이벤트 | payload | blockedWhen |
|---|---|---|
| `onItemClick` | `{ key: string; event: MouseEvent }` | `["loading"]` |

**2.0.0 파괴적 변경: 1.x 의 payload 는 `string` 이었다.**
`loading` 상태에서는 발화 금지 — 구현은 **이미** `event.preventDefault()` 로 `<a>` 기본 동작도 막는다
(`TodoCard.tsx:25-28`).

## 접근성 계약

- **role**: `region`
- **keyboard**: `Tab` · `Shift+Tab` · `Enter`
- **focusVisible**: `true`
- **aria-busy**: `loading` 일 때
- **contrastMin**: 4.5

계약 `a11y.aria`:

- `aria-labelledby`: 카드 제목 요소를 가리켜 region 에 접근성 이름을 부여한다.

## 토큰

`color.surface.default` · `color.border.default` · `radius.md` · `space.5`(padding) ·
`space.4`(gap) · `space.2`(itemGap) · `color.text.default`(title) · `typography.label.md` ·
`typography.body.md` · **`color.feedback.danger.text`(countPending)** ·
**`color.text.disabled`(countEmpty)** · `color.border.focus`(focusRing).

`countPending`/`countEmpty` 두 토큰이 위 `items` 의 분기를 시각으로 만든다.

`responsive.behavior` 는 `fluid`. 의존: `Card` · `Badge`.
