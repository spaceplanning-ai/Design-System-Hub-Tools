개발

프론트엔드 · 엔지니어

Storybook에서 컴포넌트와 토큰을 그대로 가져다 씁니다.

  1. 1Storybook에서 컴포넌트 API·상태·베리언트를 확인
  2. 2CSS 변수 var(--ds-color-primary) 로 토큰 사용 — 프리셋 전환에 자동 대응
  3. 3값 변경은 tokens/*.json 한 곳만 → pnpm build:tokens
  4. 4pnpm verify:parity 로 Figma와 100% 일치 자동 검증

기획 · 디자인

프로덕트 디자이너 · 기획

Figma 플러그인으로 클릭 몇 번에 디자인 시스템을 생성합니다.

  1. 1Figma Dev Tools 플러그인에서 메인·서브·폰트 색과 폰트를 선택
  2. 2"생성" → 컬러·타이포 페이지 + Variables + 컴포넌트 세트가 자동 생성
  3. 3고른 색이 버튼·텍스트까지 그대로 반영 — 프리셋 모드로 관리
  4. 4네이티브 베리언트·속성으로 편집, Storybook과 1:1 대응
