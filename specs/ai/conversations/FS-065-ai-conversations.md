## 1. 화면 개요

| 항목 | 내용 |
|---|---|
| 목적 | 지금까지의 에이전트 대화를 최근 갱신 순으로 보여주고, 열거나 지운다 |
| 역할(주 사용자) | 시스템 관리자 · 운영자 |
| 진입 경로 | 좌측 GNB > AI > AI 에이전트 > 대화 목록 (`nav-config.ts`, `to: '/ai/conversations'`) |

### 1.1 이 화면에 없는 기능 (요구사항)

| 없는 것 | 이유 |
|---|---|
| 대화 보관(영속) | 저장소에 쓰지 않는다. 새로고침하면 사라진다 — localStorage 에 넣으면 '저장됐다' 는 인상만 주고 다른 기기에서는 없다. **화면이 이 사실을 배너로 밝힌다**(FS-065-EL-002) |
| 일괄 선택 삭제 | 대화 수가 세션 수명에 묶여 적다. 체크박스·선택 바를 두면 쓰이지 않는 표면이 는다. 행 단위 삭제만 둔다 |
| 검색 · 필터 · 페이징 | 같은 이유. 좌측 레일의 검색(FS-064-EL-001)이 이미 있다 |
| 제목 수정 | 제목은 첫 질문에서 파생된다. 고칠 수 있게 하려면 서버가 보관해야 한다 |
