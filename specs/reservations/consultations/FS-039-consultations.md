# FS-039. 상담 예약 관리 (목록·등록/수정)

## 1. 화면 개요

| 항목 | 내용 |
|---|---|
| 목적 | 상담 예약(방문·전화·화상)을 상태·키워드로 좁혀 찾고, 고객·희망일시·담당·상태·메모를 등록·수정하며, 상태 흐름(요청→확정→상담완료/노쇼/취소)을 허용된 전이로만 옮기고, 단건·일괄 삭제한다 |
| 역할(주 사용자) | 관리자 (구현에 역할 분기 없음 — §4.1) |
| 진입 경로 | 좌측 GNB > 예약/신청 관리 > 상담 예약 (`/reservations/consultations`) |
| 포함 화면 | 목록 `/reservations/consultations` · 등록 `/reservations/consultations/new` · 수정 `/reservations/consultations/:id/edit` (`App.tsx:276-281` 확인) |
| 구현 경로 | `apps/admin/src/pages/reservations/consultations/**` + 섹션 공용 `pages/reservations/_shared/{booking,calendar,resources}.ts` |
| 대응 SCR | SCR-039 (미작성 — §7) |
